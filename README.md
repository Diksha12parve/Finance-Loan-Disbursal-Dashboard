# Finance Loan Disbursal Dashboard

## Overview
Interactive Power BI dashboard developed to analyze loan disbursal performance, branch contribution, product distribution, and customer trends.

## Tools Used
- Power BI
- DAX
- Python (Pandas)
- Excel

## Key Metrics
- Total Customers
- Total Loan Accounts
- Total Loan Amount
- Total Disbursal Amount
- Average Interest Rate
- Average Tenure

## Features
- Branch-wise Disbursal Analysis
- Product-wise Analysis
- Monthly Customer Trend
- Loan Status Analysis
- Interactive Filters and Slicers
